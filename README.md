# InsuranceChallenge

## Installation

- [Node.js](https://nodejs.org/) v12+ to run.
- [Cypress](https://docs.cypress.io/guides/getting-started/installing-cypress)

Install the dependencies.

```sh
cd InsuranceChallenge
npm i
```

## To run on Cypress App

```sh
npm run cy:open
```

## To run on Terminal

```sh
npm run cy:tests
```
